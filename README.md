# Speak — Text to Speech App

A clean, minimalist **Text to Speech** web app built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just open `index.html` in any modern browser.

## Features

- Speak / Pause / Resume / Stop controls
- Voice selector — loads all installed system voices
- Speed, Pitch & Volume sliders
- Animated progress bar
- `Ctrl+Enter` keyboard shortcut to speak
- Character counter (up to 5000 chars)
- Works in Chrome, Edge, Firefox, Safari & Opera
- Fully offline after first load

## Usage

Open `index.html` in your browser. No build step required.
